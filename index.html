Q .4 What is execution context?

The Execution Context contains the code that's currently running, and everything that aids in its execution. During the Execution Context run-time, the specific code gets parsed by a parser, the variables and functions are stored in memory, executable byte-code gets generated, and the code gets executed.

Q .5.What is meant by first class function?

In any Programming language, when a function is treated as a variable then it is called as First class Function.

  • we can assign a function to a variable

  • We can pass a function as an argument in other functions.

  • returning a function

  • Arrow Functions etc..,Examples are:
    •                 1.Function to a variable:
                let myfunction=function(){
                    console.log("hi");
                }
                myfunction();

                2.function as a argument:
                function greet(){
                    console.log("Happy")
                }
                function specialgreet(callback,message){
                    console.log(callback(),message)
                }
                specialgreet(greet,"Hello friend");

                3.return a function:
                function outer(){
